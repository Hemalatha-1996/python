## 📌 Project Overview

The Agriculture Production Dashboard is an interactive data visualization web application built using **Python and Streamlit**.
It provides insights into India's agricultural production trends across different crops, states, and years through dynamic charts and analytics.

The dashboard helps users explore crop production patterns, compare states, analyze trends, and understand agricultural growth over time.

---

## 🚀 Features

* 📊 Crop-wise production analysis
* 📈 Year-wise production trends
* 🗺️ State-wise agriculture insights
* 🌾 Rice, Wheat, Sorghum, Millet analysis
* 📉 Trend and comparison charts
* 📌 Interactive visualizations
* ⚡ Fast and responsive Streamlit UI

## ▶️ How to Run Locally

### Step 1: Clone Repository

```
git clone https://github.com/your-username/agriculture-dashboard.git
```

### Step 2: Go to Project Folder

```
cd agriculture-dashboard
```

### Step 3: Install Requirements

```
pip install -r requirements.txt
```

### Step 4: Run Streamlit App

```
streamlit run agriculture_dashboard.py
```
